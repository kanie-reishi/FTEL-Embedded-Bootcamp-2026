# FTEL Embedded Bootcamp 2026

Welcome to my progress repository for the **FTEL Embedded Systems Bootcamp 2026**!

## 🎯 Course Overview

This bootcamp is a comprehensive journey designed to build strong foundations in embedded systems software engineering. The curriculum starts from fundamental problem-solving and algorithmic thinking, eventually scaling up to complex, low-level system design. 

The ultimate goal of this bootcamp is to understand the core principles of embedded architectures and **develop a Real-Time Operating System (RTOS) from scratch**.

## 🚀 Learning Path Roadmap

The repository is structured to reflect my progression through the following core phases of the bootcamp:

### Phase 1: Algorithmic Foundations (Current)
- **Objective:** Strengthen foundational logic, problem-solving, and efficient C/C++ coding practices.
- **Content:** Data structures, algorithms, bit manipulation, and optimization techniques.
- **Directory:** `/Leetcode`

### Phase 2: Embedded C & Microcontroller Basics
- **Objective:** Transition from generic programming to hardware-specific coding.
- **Content:** Memory management (pointers, volatile, registers), interrupt handling, timers, GPIOs, and peripheral communication (UART, I2C, SPI).

### Phase 3: Hardware Architecture
- **Objective:** Understand what happens under the hood.
- **Content:** ARM Cortex-M architecture, boot sequence, memory maps, system clock configuration, and basic assembly language.

### Phase 4: RTOS Concepts & Development
- **Objective:** The capstone project of the bootcamp.
- **Content:** 
  - Concurrency vs. Parallelism
  - Context Switching & the scheduler
  - Tasks, States, and Control Blocks (TCBs)
  - Synchronization primitives (Semaphores, Mutexes, Queues)
  - Memory isolation and resource management
